SProjectile

A projectile system for the use in the Unity Engine, featuring multiple different types for accurate interactions with the rest of the game world.

What is it?

SProjectile is a versatile system for simulating projectiles in Unity projects, all at various speeds that can interact with the same target. Various types of projectiles exist, each having their own pros and cons.

  • Documentation

Using SProjectile

SProjectile can be incorporated into any project with ease simply by installing the package. The documentation can assist in picking the right type of projectile system to use in your project. The package can be downloaded from here or the Unity Asset Store (Pending).

  • Download

About SProjectile

Originally created for use in Carbon Vanguard, which has multiple different types of projectiles all travelling at various speeds. Simple rigidbodies weren't suiting the required needs
